# IoT Communication Protocols

A comprehensive collection of Internet of Things (IoT) communication protocol implementations developed as part of embedded systems and IoT laboratory experiments. This repository demonstrates the implementation, configuration, and performance of widely used communication protocols for connecting embedded devices to local and cloud-based systems.

---

## Overview

Modern IoT systems rely on efficient communication protocols to exchange data between sensors, embedded devices, cloud platforms, and users. This repository explores several industry-standard protocols through practical implementations using embedded hardware.

The experiments focus on understanding protocol architecture, communication methods, reliability, latency, and real-world IoT applications.

---

## Protocols Implemented

### MQTT (Message Queuing Telemetry Transport)

- Publish/Subscribe communication model
- Lightweight messaging protocol
- Broker-based communication
- Sensor data transmission
- Real-time IoT applications

---

### TCP (Transmission Control Protocol)

- Reliable communication
- Connection-oriented protocol
- Error checking and retransmission
- Ordered packet delivery
- Client-Server communication

---

### UDP (User Datagram Protocol)

- Connectionless communication
- Low latency
- Fast data transmission
- Minimal overhead
- Real-time applications

---

### Bluetooth Communication

- Short-range wireless communication
- Device pairing
- Serial communication
- Embedded device control
- Wireless sensor connectivity

---

### ThingSpeak Cloud Integration

- IoT cloud platform
- Sensor data visualization
- Real-time monitoring
- Cloud analytics
- Remote data storage
